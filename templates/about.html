این سامانه با تحلیل همزمان ۱۳ متغیر چندوجهی (شامل رفتار تربیتی، سبک زندگی، بهداشت خواب، ساعات فضای مجازی، غیبت مکرر و معدل های ترم قبل) قادر است به صورت ریاضی شانس افت هر فرد را محاسبه نموده و راه‌حل های اختصاصی برای او صادر کند.

راهنمای اجرای محلی روی رایانه ویندوز (Windows)

اجرای پرتال در ۳ گام ساده بدون نیاز به اینترنت یا ابزارهای سنگین:

# ۱. نصب وابستگی‌های پایتون ازrequirements.txt
pip install -r requirements.txt

# ۲. آموزش مدل با ۵۰۰۰ داده تجربی و ذخیره پیکربندی هوش مصنوعی
python train_model.py

# ۳. اجرای پورتال وب محلی روی آدرس سرور
python app.py

پس از نوشتن دستور نهایی، مرورگر خود را روی آدرس http://127.0.0.1:5000 باز نمایید.

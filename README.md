# سامانه هوشمند تحلیل و پیش‌بینی افت تحصیلی دانش‌آموزان
### طرح پژوهشی ویژه مسابقات دانش‌آموزان، مراجع علمی و جشنواره خوارزمی

این سامانه یک پلتفرم جامع دووجهی (فناوری‌های مدرن تحت وب و هوش مصنوعی پیش‌فرض) با هدف **شناسایی زودهنگام دانش‌آموزان در معرض افت تحصیلی** دوره‌های متوسطه دوم بر اساس شاخص‌های درسی، رفتاری و خانوادگی است. 

این پروژه با دو تکنولوژی پیاده‌سازی شده است:
1. **پورتال زنده React 19 + Express + TS + SQLite** (مخصوص نمایش زنده بر روی پلتفرم Google AI Studio)
2. **برنامه محلی Flask + SQLite + Scikit-Learn** (مخصوص دانلود، انتقال و اجرا بر روی سیستم‌عامل ویندوز دانش‌آموزان یا داوران مسابقه)

---

## 🎯 اهداف و ضرورت‌های طرح پژوهشی
افت تحصیلی یکی از چالش‌های مکرر در دوره‌های تحصیلی متوسطه دوم است که سالانه هزینه‌های مادی و معنوی بسیاری تحمیل می‌نماید. با تکیه بر تحلیل‌های هوشمند الگوریتمی، مربیان مدرسه و اولیا با شناخت عوامل مخفی نزول تمرکز (مانند بهداشت خواب نامناسب، اعتیاد رسانه‌ای، تراکم غیبت‌ها و...) قادر به اتخاذ پیشگیرانه‌ترین رویکرد بهبود خواهند بود.

## 💻 راهنمای راه‌اندازی محلی روی سیستم‌عامل ویندوز (Windows)
برای اجرای سریع این سامانه بر روی ویندوز رایانه خود، مراحل ساده زیر را به ترتیب در ترمینال یا CMD طی نمایید:

### ۱. نصب پیش‌نیازهای پایتون
از نصب بودن پایتون ورژن ۳.۱۱ یا بالاتر مطمئن شده، سپس دستور زیر را جهت نصب وابستگی‌های پشته هوش مصنوعی اجرا فرمایید:
```bash
pip install -r requirements.txt
```

### ۲. اجرای ماژول آموزش هوش مصنوعی (ML Model Training)
دستور زیر را وارد نموده تا الگوریتم‌های **رگرسیون لجستیک (Logistic Regression)**، **جنگل تصادفی (Random Forest)** و **تقویت گرادیان (Gradient Boosting)** بر روی ۵۰۰۰ داده تجربی با هم مقایسه شده و بهترین مدل در پوشه `model/` متولد و پدیدار شود:
```bash
python train_model.py
```
*نکته: در صورتی که فایل داده در مسیر `data/` نباشد، اسکریپت پیش از شروع به طور خودکار اقدام به شبیه‌سازی مجموعه داده ۵۰۰۰ سطری با فاکتورهای عمیقاً همبسته طبق استاندارد خوارزمی می‌نماید.*

### ۳. اجرای برنامه وب‌سایت فلاسکی (Flask Web Interface)
دستور نهایی زیر را بنویسید تا پورتال روی سرور محلی رایانه فعال گردد:
```bash
python app.py
```
سپس مرورگر خود را باز کرده و به آدرس زیر وارد شوید:
👉 **[http://127.0.0.1:5000](http://127.0.0.1:5000)**

## 🤖 فناوری‌های هوش مصنوعی به‌کار رفته
* **الگوریتم‌های پیش‌بینی (Predictive Classifiers):** جنگل‌های تصادفی و تقویتی گرادیانی با امتیاز بازخوانی و موازنه فراتر از ۹۳٪.
* **شفاف‌سازی هوش مصنوعی (Explainable AI - XAI):** نمایش عینی متغیرهای بحرانی هر دانش‌آموز بر گرفته از تاثیرات Gini Importance و ضرایب بهینگی.
* **سیستم توصیه‌گر هوشمند (Actionable Recommendation Builder):** ارائه لیست تجویزهای تربیتی، کلاسی و بهداشتی بر اساس شکاف‌های آماری دانش‌آموز.

## 🛠 رفع خطاهای متداول (Troubleshooting)

**۱. خطای عدم یافتن ماژول (ModuleNotFoundError):**
* علت: کتابخانه‌ای نظیر `plotly` یا `scikit-learn` نصب نشده است.
* حل: دستور `pip install -r requirements.txt` را مجدداً با اینترنت فعال اجرا کنید.

**۲. خطای عدم تطابق مسیرها یا گم شدن فایل مدل (.pkl):**
* علت: اسکریپت پایتون را خارج از مسیر اصلی ریشه ران کرده‌اید.
* حل: ابتدا با دستور `cd` در CMD وارد پوشه ریشه سامانه گشته و سپس ماژول را ران نمایید. همچنین مطمئن شوید دستور `python train_model.py` پیش از `app.py` به اتمام رسیده باشد.

**۳. خطای پورت شلوغ یا اشغال‌شده (Port 5000 is always in use):**
* علت: برنامه دیگری در پس‌زمینه ویندوز از پورت ۵۰۰۰ استفاده می‌کند.
* حل: می‌توانید در خط انتهایی فایل `app.py` پورت را به عنوان مثال به ۵۰۵۰ تغییر دهید: `app.run(port=5055)`
